# Steam8 Poster — セットアップガイド

Steamゲーム布教まとめ画像を自動生成する Webアプリです。  
**API キーや課金は一切不要**です。Steam の公開 API を使用しています。

## 方法 A：ローカルで動かす（推奨）

### 1. Python のインストール確認

ターミナル（PowerShell / Terminal）を開いて以下を実行してください。

```
python --version
```

`Python 3.10.x` 以上が表示されれば OK です。  
未インストールの場合は [python.org](https://www.python.org/downloads/) からダウンロードしてください。  
インストール時に **「Add Python to PATH」にチェックを入れること**を忘れずに。

---

### 2. ファイルの準備

受け取った ZIP ファイルを展開するか、Git でクローンします。

**ZIP の場合**

1. ZIP を任意のフォルダに展開する
2. ターミナルでそのフォルダを開く

```
cd C:\Users\あなたの名前\Downloads\App_035_Steam8Poster
```

**Git の場合**

```
git clone https://github.com/[リポジトリURL].git
cd App_035_Steam8Poster
```

---

### 3. 仮想環境の作成（任意・推奨）

他のプロジェクトと依存関係が混ざらないよう、仮想環境を作ることを推奨します。

```
python -m venv .venv
```

仮想環境を有効化します。

**Windows（PowerShell）**
```
.venv\Scripts\Activate.ps1
```

**Mac / Linux**
```
source .venv/bin/activate
```

> 有効化すると `(.venv)` がプロンプトの先頭に表示されます。

---

### 4. 必要なライブラリのインストール

```
pip install -r requirements.txt
```

インストールされるのは以下の 4 つだけです。

| ライブラリ | 用途 |
|---|---|
| streamlit ≥ 1.37.0 | Web UI フレームワーク（ダイアログ機能に必要） |
| Pillow | 画像生成 |
| requests | Steam API 通信 |
| streamlit-sortables | スロットのドラッグ＆ドロップ並べ替え |

---

### 5. アプリの起動

```
streamlit run app.py
```

数秒後にブラウザが自動で開きます。  
開かない場合は `http://localhost:8501` をブラウザで開いてください。

---

### 6. 初回起動時のフォントダウンロード

**初回のみ**、日本語フォント（Noto Sans CJK JP Bold）を GitHub から自動ダウンロードします。  
完了まで約 30 秒かかります。画面上にスピナーが表示されている間はそのままお待ちください。

ダウンロードが完了すると `NotoSansCJKjp-Bold.otf` がフォルダ内に保存されます。  
次回以降の起動ではダウンロードは発生しません。

---

### 7. アプリの終了

ターミナルで `Ctrl + C` を押すと停止します。

## 方法 B：Streamlit Cloud で動かす（PCスペック不問）

Streamlit が提供する無料のクラウドサービスにデプロイする方法です。  
GitHub アカウントがあれば、インストール作業なしでブラウザだけで使えます。

### 1. GitHub にリポジトリを作成

1. [github.com](https://github.com) にログイン
2. 右上の「+」→「New repository」
3. 名前を `steam8-poster` などにして、**Public** を選択して作成

---

### 2. ファイルをアップロード

受け取ったフォルダの中身を GitHub にアップロードします。  
Git に慣れていない場合は「Upload files」ボタンから以下のファイルをドラッグ＆ドロップしてください。

アップロードが必要なファイル:

```
app.py
requirements.txt
.streamlit/config.toml
```

> `.gitignore` や `CONTEXT.md` はなくても動作します。  
> `NotoSansCJKjp-Bold.otf` はアップロード不要（起動時に自動取得）。

---

### 3. Streamlit Cloud でデプロイ

1. [share.streamlit.io](https://share.streamlit.io) にアクセス
2. GitHub アカウントでサインイン
3. 「New app」をクリック
4. 以下を設定して「Deploy」

| 項目 | 設定値 |
|---|---|
| Repository | 作成したリポジトリ名 |
| Branch | `main` |
| Main file path | `app.py` |

5. デプロイ完了まで 2〜5 分待つ
6. 表示された URL をブックマークすれば完了

> 無料プランでは一定時間アクセスがないとスリープします。  
> 再アクセス時に「This app has gone to sleep...」と表示されたら  
> 「Wake up app」ボタンを押して 1〜2 分待ちます。

## アプリの使い方

### 全体設定

アプリ上部の「⚙️ 全体設定」を開いて設定します。

| 設定項目 | 説明 |
|---|---|
| 全体見出しを表示する | ON: 8本紹介（ヘッダーあり）/ OFF: 10本紹介（ヘッダーなし） |
| 全体見出し | ポスター上部に表示するタイトル文字（最大40文字） |
| テーマカラー | 5種類のデザインテーマから選択 |
| 背景スタイル | ぼかし背景（ゲームのアート画像を使用）or 単色背景 |
| ぼかし強度 | ぼかし背景選択時のみ表示（1〜40） |

### ゲームの登録

#### 手動で1本ずつ登録する場合

1. スロットカードの **「✏️ 編集」** ボタンをクリック（ポップアップが開く）
2. ゲームタイトルを入力して **Enter キー** または **「🔍」** ボタンで検索
3. 右側にサムネプレビューが表示されるので、目的のゲームを選んで **「✅ このゲームに決定」**
4. レビュー文（140文字まで・絵文字OK）とプレイ人数を入力
5. **「💾 保存して閉じる」** をクリック

> レビュー文は入力中にリアルタイムで「N / 140 文字」と表示されます。  
> 140字を超えると赤くなり、保存ボタンが無効になります。

#### ウィッシュリストから一括インポートする場合

1. **「📥 ウィッシュリストからインポート」** を開く
2. Steam のウィッシュリスト URL を入力して **「取得」**  
   例: `https://store.steampowered.com/wishlist/id/ユーザー名/`
3. 取得できたゲーム一覧からインポートするものを選ぶ
4. **「📥 選択したゲームをスロットに追加」** をクリック

> ウィッシュリストは**公開設定**になっている必要があります。  
> API キーは不要です。

#### スロットの並べ替え

1. スロットヘッダー右端の **「🔀 並び替え」** ボタンをクリック
2. 並び替えモードに切り替わるので、ドラッグ＆ドロップで順序を変更
3. **「✅ 並び替え完了」** ボタンを押すと通常グリッドに戻ります

### ポスターの生成・ダウンロード

1. 1本以上ゲームを登録したら **「🎨 ポスターを生成する」** をクリック
2. 生成完了後、プレビューが表示されます
3. **「⬇️ PNG でダウンロード」** で保存

> ダウンロードされるファイル名の例:  
> `steam_8pick_20260410.png`（8本モード）  
> `steam_10pick_20260410.png`（10本モード）  
>
> 設定を変えて再生成したい場合はそのまま **「🔄 ポスターを再生成する」** をクリックします。
