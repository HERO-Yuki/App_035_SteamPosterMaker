# SteamPosterMaker — よくあるトラブルと対処法

---

## 起動・インストール

### `streamlit: command not found` と表示される

Streamlit がインストールされていないか、PATH が通っていません。

```
pip install -r requirements.txt
```

を再実行してください。仮想環境を使っている場合は、有効化されているか確認してください。

```
# Windows
.venv\Scripts\Activate.ps1

# Mac / Linux
source .venv/bin/activate
```

---

### `pip install` 時に権限エラーが出る

管理者権限なしにシステム全体へのインストールを試みている可能性があります。  
仮想環境を使うか、`--user` オプションを付けて実行してください。

```
pip install --user -r requirements.txt
```

---

### ブラウザが自動で開かない

`http://localhost:8501` を手動でブラウザに入力してください。  
ポートが別の番号（例: 8502）になっている場合は、ターミナルに表示されている URL を確認してください。

```
  You can now view your Streamlit app in your browser.
  Local URL: http://localhost:8501   ← この URL を使う
```

---

### `The execution policy...` エラーが出る（Windows）

PowerShell のスクリプト実行ポリシーが制限されています。以下のコマンドを実行してから再試行してください。

```
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

## フォント・日本語表示

### 起動時にフォントダウンロードが始まらない / 失敗する

- インターネット接続を確認してください
- 企業・学校ネットワークではファイアウォールで GitHub へのアクセスがブロックされる場合があります
- フォントダウンロードに失敗してもアプリ自体は動作しますが、ポスター上の日本語テキストが正しく表示されない場合があります

**手動でフォントを配置する方法**

1. 以下のリンクからフォントを手動ダウンロードする  
   [NotoSansCJKjp-Bold.otf（GitHub）](https://github.com/googlefonts/noto-cjk/raw/main/Sans/OTF/Japanese/NotoSansCJKjp-Bold.otf)
2. ダウンロードしたファイルを `app.py` と同じフォルダに置く
3. アプリを再起動する

---

### ポスターの文字が豆腐（□）になる

フォントが読み込まれていない状態です。上記「手動でフォントを配置する方法」を試してください。

## Steam API・ゲーム検索

### 検索してもゲームが見つからない

- タイトルの一部だけで検索してみてください（例: `Elden` で検索）
- 日本語・英語どちらでも検索できます
- Steam のサーバーが一時的にダウンしている場合があります。数分後に再試行してください

---

### 価格が「不明」と表示される

以下のいずれかが原因です。

- **年齢制限コンテンツ**: Steam API が成人向けコンテンツの詳細を返さない仕様のため、価格取得ができない場合があります
- **地域限定販売**: 日本向けに販売されていないゲームは価格情報がありません
- **無料プレイ以外の特殊な価格体系**: サブスクリプション型などは対応していません

いずれの場合もポスター生成自体は正常に行えます。

---

### 検索結果にサムネイルが表示されない

Steam CDN サーバーへの接続に失敗している可能性があります。  
ページを更新してから再度検索してみてください。

## ポスター生成

### 生成ボタンを押すと「画像の生成に失敗しました」と表示される

以下を確認してください。

1. **フォントファイルが存在するか**: `app.py` と同じフォルダに `NotoSansCJKjp-Bold.otf` があるか確認
2. **メモリ不足**: 1920×1080 の画像生成には一定のメモリが必要です。他のアプリを閉じて再試行
3. **ゲーム画像の取得失敗**: 一度スロットをクリアして登録しなおしてください

---

### ポスターを生成した後、設定を変えたら消えてしまった

現在の仕様として、設定変更のたびにページが再描画されポスターが消えます。  
**先にダウンロードしてから**設定を変更してください。

## Streamlit Cloud

### デプロイ後に「Module not found」エラーが出る

`requirements.txt` がリポジトリにアップロードされているか確認してください。

---

### アプリが「This app has gone to sleep...」と表示される

無料プランでは一定時間（約 7 日間）アクセスがないとスリープします。  
画面の「Wake up app」ボタンをクリックして 1〜2 分お待ちください。

---

### デプロイ後にフォントが取得されない

Streamlit Cloud の環境では GitHub からのダウンロードが正常に行われるはずですが、  
もし日本語が表示されない場合は App の「Manage app」→「Reboot app」で再起動してみてください。

---

## それでも解決しない場合

アプリ作成者にターミナルのエラーメッセージをそのままコピーして送ってください。  
`streamlit run app.py` 実行時のログがデバッグに役立ちます。
