# App_035 SteamPosterMaker

Steam ゲーム布教用まとめ画像（最大10本紹介）を X (Twitter) 向けに **1920×1080 PNG** で自動生成する Streamlit アプリ。

---

## 機能

| カテゴリ | 内容 |
|----------|------|
| ゲーム検索 | キーワード入力 → Enter キーまたは🔍ボタンで Steam API 検索。結果上位10件＋サムネプレビュー付き |
| スロット管理 | 最大10スロット（2×N グリッド）。ポップアップダイアログで編集・クリア |
| 並べ替え | 「🔀 並び替え」ボタンでモード切替 → ドラッグ＆ドロップで順序変更（`streamlit-sortables`）|
| 全体見出し | ON: ヘッダー120px + 8スロット（2×4）／OFF: ヘッダーなし + 10スロット（2×5）。常に1920×1080出力 |
| 一括インポート | Steam ウィッシュリストの URL を貼るだけで自動取得・スロット仮置き（API キー不要） |
| 価格表示 | セール: `-50% ¥500` / 無料: `無料` / 取得不可: `不明` をサムネ右下バッジで表示 |
| 年齢制限対応 | API が詳細を返さないゲームは錠前アイコン（Pillow描画）+ "18+" で可視化。UI には🔞バッジ表示 |
| テーマカラー | 5テーマ（後述）|
| 背景スタイル | サムネぼかし（強度スライダー）/ 単色 の2択 |
| レビュー文 | 最大140文字（絵文字対応）。入力中にリアルタイム文字数カウンター表示。超過時は保存不可 |
| 全体見出し | 最大25文字（64pt フォントと 1920px ヘッダー幅に基づく上限）|
| プレイ人数 | プリセットから複数選択（ソロ / ローカル協力 / ローカル対戦 / オンライン協力 / オンライン対戦 / MMO / その他）|
| 出力 | 1920×1080 PNG プレビュー＆ダウンロード。ウォーターマーク付き |

## 画像レイアウト（1920 × 1080）

### 全体見出し ON（8本紹介）

```
┌──────────────────────────────── 1920px ────────────────────────────────┐
│              全体見出し（ヘッダー 120px）                               │ 120px
├─────────────────────────────────────────────────────────────────────────┤
│ [サムネ 380px][│][タイトル]         [サムネ 380px][│][タイトル]         │
│               [│][👥 人数]                         [│][👥 人数]         │ 215px
│               [│][💬 レビュー文…]                  [│][💬 レビュー文…]  │
├─────────────────────────────────────────────────────────────────────────┤
│                            3〜4 行（同レイアウト）                       │ 215px × 3
└─────────────────────────── [Generated by SteamPosterMaker] ────────────┘
```

### 全体見出し OFF（10本紹介）

```
┌──────────────────────────────── 1920px ────────────────────────────────┐
│ [サムネ 380px][│][タイトル]         [サムネ 380px][│][タイトル]         │
│               [│][👥 人数]                         [│][👥 人数]         │ 192px × 5行
│               [│][💬 レビュー文…]                  [│][💬 レビュー文…]  │
│                            計10枚（同レイアウト）              [WM]     │
└────────────────────────────────────────────────────────────────────────┘
```

- カードサイズ: **954 × 235 px**（見出しON）/ **954 × 211 px**（見出しOFF）
- カード間マージン: **4 px**（縦横共通）
- サムネ幅: **380 px**（カード左端。Steam ヘッダー画像 460×215 の約 83% を表示）
- アクセント縦区切り線（3px）がサムネとテキストを分離
- 価格バッジ: サムネ右下に半透明オーバーレイ。フォント **24pt**・端から **10px** の余白
- テキスト幅: `954 - 380 - 3 - 12 - 12 = 547 px`（日本語ピクセル幅ベースで自動折り返し）

## インストール

```bash
pip install -r requirements.txt
```

## 起動

```bash
streamlit run app.py
```

## 初回起動時の注意

日本語フォント（Noto Sans CJK JP Bold）を GitHub から自動ダウンロードします（約30秒）。  
カレントディレクトリに `NotoSansCJKjp-Bold.otf` として保存されます。  
ダウンロード失敗時はシステムフォントにフォールバックします（日本語表示が乱れる場合があります）。

---

## Streamlit Community Cloud へのデプロイ

1. このフォルダを GitHub リポジトリとして push
2. [share.streamlit.io](https://share.streamlit.io) で `app.py` を指定してデプロイ
3. `requirements.txt` が自動で読み込まれます（API キー不要）

詳細は [`docs/setup-guide.md`](docs/setup-guide.md) を参照してください。

## 技術スタック

| 用途 | ライブラリ |
|------|-----------|
| Web UI・状態管理 | Streamlit ≥ 1.37.0 |
| ポップアップダイアログ | `st.dialog`（Streamlit 1.37+ 必須）|
| ドラッグ＆ドロップ | streamlit-sortables |

| データ源 | Steam Web API（認証不要・公開エンドポイント）|
| 日本語フォント | Noto Sans CJK JP Bold（起動時に自動取得）|
