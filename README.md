# App_035 Steam8 Poster

Steamゲーム布教用まとめ画像（8本紹介）を X (Twitter) 向けに自動生成する Webアプリ。

## 機能

- **Steam ゲーム検索**: タイトルを入力するだけで Steam ストア API から候補を取得（結果上位10件表示）
- **8本スロット管理**: 最大8本のゲームを登録。未入力スロットはグレープレースホルダーで描画
- **価格自動取得**: セール中は `-50%  ¥500` のように整形。無料ゲームは「無料」表示。年齢制限ゲーム・API障害時もクラッシュしない
- **レビュー文**: 最大140文字（絵文字対応、Python `len()` でカウント）
- **プレイ人数プリセット（複数選択）**: ソロ / ローカル協力 / ローカル対戦 / オンライン協力 / オンライン対戦 / MMO / その他
- **5つのテーマカラー**: ゲームジャンル別にデザイン（後述）
- **背景スタイル2種**: サムネ拡大ぼかし（強度スライダー付き）or 単色
- **1920×1080 PNG 出力**: ウォーターマーク `Generated by Steam8 Poster` 付き、ダウンロードボタン完備
- **全設定をセッション間保持**: `st.session_state` でページ再読込後もデータ消失なし

## テーマカラー

| テーマ | ターゲットジャンル | 背景色 | アクセント |

## インストール

```bash
pip install -r requirements.txt
```

## 起動

```bash
streamlit run app.py
```

## 初回起動時の注意

日本語フォント（Noto Sans CJK JP Bold）を GitHub から自動ダウンロードします（約30秒）。  
カレントディレクトリに `NotoSansCJKjp-Bold.otf` として保存されます。  
ダウンロード失敗時はシステムフォントにフォールバックします（日本語表示が乱れる場合があります）。

## Streamlit Community Cloud へのデプロイ

1. このフォルダを GitHub リポジトリとして push
2. [share.streamlit.io](https://share.streamlit.io) で `app.py` を指定してデプロイ
3. `requirements.txt` が自動で読み込まれます（APIキー不要）

│              [│]   [価格]   [人数]                   [│]   [価格]   [人数]                                     │ 215px

├────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│                                     03-04 行（同レイアウト）                                                    │ 215px
├────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│                                     05-06 行（同レイアウト）                                                    │ 215px
├────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│                                     07-08 行（同レイアウト）                                            [WM]   │ 215px

- カードサイズ: **930 × 215 px**（マージン 20px を差し引いて均等分割）
- サムネイル: **215 × 215 px** の正方形クロップ（カード左端）
- テキスト幅: **688 px**（日本語ピクセル幅ベースで自動折り返し）
