# SteamPosterMaker

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://steam-poster-maker.streamlit.app)
![Python](https://img.shields.io/badge/Python-3.10%2B-blue)
![License](https://img.shields.io/badge/License-MIT-green)

**Steam でハマったゲームを布教したい人向けのツールです。**  
紹介したいゲームを最大10本選ぶだけで、X（Twitter）にそのまま投稿できる高画質まとめポスター（1920×1080 PNG）を自動生成します。

**[アプリを試す → steam-poster-maker.streamlit.app](https://steam-poster-maker.streamlit.app)**

---

![OGP](ogp.png)

---

## 使い方

| STEP | 操作 |
|------|------|
| 1 | ゲームスロットの「編集」ボタンを押す |
| 2 | タイトルを検索（キーワード / AppID / Steam ストア URL に対応）|
| 3 | ゲームを選んでレビュー文を入力し「保存」 |
| 4 | 必要に応じてテーマ・背景スタイルを変更 |
| 5 | 「ポスターを生成する」→ PNG をダウンロード → X に投稿 |

> ゲームは最大10本。空のスロットは「空欄カード」としてポスターに自動配置されます。

---

## 出力イメージ（1920 × 1080 px）

```
┌──────────────────────── 1920px ────────────────────────┐
│             全体見出し（任意・最大25文字）               │
├────────────────────────────────────────────────────────┤
│ [サムネ] ┃ ゲームタイトル      [サムネ] ┃ ゲームタイトル │
│          ┃ レビュー文…                  ┃ レビュー文…   │
├────────────────────────────────────────────────────────┤
│                  … 最大10本分（同レイアウト）           │
└─────────────── Generated by SteamPosterMaker ──────────┘
```

- カードサイズ: **954 × 235 px**（見出し ON・8本）/ **954 × 192 px**（見出し OFF・10本）
- サムネ: contain モード（アスペクト比保持・余白は黒ベタ）
- レビュー文: 日本語ピクセル幅ベースで自動折り返し。枠に収まらない場合はフォントサイズを自動縮小

## 主な機能

### ゲーム入力

| 機能 | 説明 |
|------|------|
| キーワード検索 | 日本語・英語タイトルで Steam API 検索。上位10件＋サムネプレビュー付き |
| AppID 直接入力 | 検索欄に数字のみ入力すると AppID として直接取得 |
| Steam URL 入力 | ストアページの URL を貼るだけでゲームを自動取得 |
| ドラッグ＆ドロップ並べ替え | 「並び替え」モードでスロットの順序を変更 |
| 全スロット一括クリア | 確認ダイアログ付きのリセット機能 |

### ポスターデザイン

| 機能 | 説明 |
|------|------|
| ゲーム数 | 8本 / 10本 を選択（カードサイズが自動調整） |
| 全体見出し | ON/OFF 切り替え可能。最大25文字 |
| テーマカラー | 5種類（下表参照） |
| 背景スタイル | ぼかし（強度スライダー）/ 単色 の2択 |
| 価格バッジ | サムネ右下に自動表示。非表示切り替えも可能 |
| 年齢制限対応 | 画像取得不可タイトルは錠前アイコン（18+）で可視化 |

### その他

| 機能 | 説明 |
|------|------|
| 言語切り替え | 画面右上のトグルで 日本語 / English を切り替え |
| X シェア | ダウンロード後にワンクリックでシェア投稿画面を開く |
| ウォーターマーク | フッターに `Generated by SteamPosterMaker` と URL を自動挿入 |

## ローカルでの開発・実行

### 必要環境

- Python 3.10 以上

### セットアップ

```bash
git clone https://github.com/HERO-Yuki/App_035_SteamPosterMaker.git
cd App_035_SteamPosterMaker
pip install -r requirements.txt
streamlit run app.py
```

### 初回起動時の注意

日本語フォント（Noto Sans CJK JP Bold）を自動ダウンロードします（初回のみ、約30秒）。  
ダウンロード先は **GitHub 公式リポジトリ → jsDelivr CDN** の順にフォールバックするため、GitHub 障害時も高確率で取得できます。  
全 URL 失敗時はシステムフォントで代替します（日本語の表示が乱れることがあります）。

### ディレクトリ構成

```
App_035_SteamPosterMaker/
├── app.py                   # メイン（Streamlit アプリ全体）
├── requirements.txt
├── README.md
├── ogp.png                  # GitHub ソーシャルプレビュー用画像
├── .gitignore
├── .streamlit/
│   └── config.toml          # Steam ライクな UI カラーテーマ
└── NotoSansCJKjp-Bold.otf   # 初回起動時に自動ダウンロード（.gitignore 対象）
```

## Streamlit Community Cloud へのデプロイ

1. このリポジトリを GitHub に push（`docs/` フォルダはローカル専用のため .gitignore 済み）
2. [share.streamlit.io](https://share.streamlit.io) で `app.py` を指定してデプロイ
3. `requirements.txt` が自動で読み込まれます（API キー・シークレット不要）
