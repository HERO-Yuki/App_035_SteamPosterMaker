# SteamPosterMaker

Steam でハマったゲームを布教したい人向けのツールです。紹介したいゲームを最大10本選ぶだけで、X（Twitter）にそのまま投稿できる高画質まとめポスター（1920×1080 PNG）を自動生成します。

---

## 機能

| カテゴリ | 内容 |
|----------|------|
| ゲーム検索 | キーワード入力 → Enter キーで Steam API 検索。上位10件＋サムネプレビュー付き |
| AppID / URL 直接入力 | 検索フォームに AppID（数字）または Steam ストア URL を貼り付けると直接取得 |
| スロット管理 | 最大10スロット（2×N グリッド）。ポップアップダイアログで編集・クリア |
| 全スロット一括クリア | 「全クリア」ボタン → 確認ダイアログでまとめて削除 |
| ダイアログ編集 | 「検索フェーズ」→「編集フェーズ」の2段階 UI。左=ゲーム画像 / 右=レビュー入力の2カラムレイアウト |
| 並べ替え | 「並び替え」ボタンでモード切替 → ドラッグ＆ドロップで順序変更 |
| 全体見出し | ON: ヘッダー付き（8〜10スロット）/ OFF: フルグリッド（10スロット）。常に 1920×1080 出力 |
| 価格表示 | セール: `-50% ¥500` / 無料: `無料` / 取得不可: `不明` をサムネ右下バッジで表示。非表示にも対応 |
| 年齢制限対応 | API が詳細を返さないゲームは錠前アイコン（Pillow 描画）+ "18+" で可視化 |
| テーマカラー | 5テーマ＋カラースウォッチ（bg / accent / card_bg 3色プレビュー） |
| 背景スタイル | サムネぼかし（強度スライダー）/ 単色 の2択 |
| レビュー文 | 最大140文字相当。リアルタイム文字数カウンター付き。改行もポスターに反映 |
| 出力 | 1920×1080 PNG プレビュー＆ダウンロード。ファイル名にポスタータイトルを含む |
| 言語 | 画面右上のトグルで日本語 / English を切り替え可能 |

## 画像レイアウト（1920 × 1080）

```
┌──────────────────────────────── 1920px ───────────────────────────────┐
│                    全体見出し（ヘッダー）                               │
├───────────────────────────────────────────────────────────────────────┤
│ [サムネ 380px] ┃ [タイトル]          [サムネ 380px] ┃ [タイトル]      │
│                ┃ [💬 レビュー文…]                   ┃ [💬 レビュー文…]│
├───────────────────────────────────────────────────────────────────────┤
│                       3〜4 行（同レイアウト）                          │
└──────────────────────── [Generated by SteamPosterMaker] ──────────────┘
```

- カードサイズ: **954 × 235 px**（見出し ON・8本）/ **954 × 192 px**（見出し OFF・10本）
- サムネ描画: contain モード（アスペクト比保持・余白は黒ベタ）
- 価格バッジ: サムネ右下に半透明オーバーレイ。フォント 24pt
- レビュー文: 日本語ピクセル幅ベースで自動折り返し・フォントサイズ自動縮小

## インストール

```bash
pip install -r requirements.txt
```

## 起動

```bash
streamlit run app.py
```

## 初回起動時の注意

日本語フォント（Noto Sans CJK JP Bold）を自動ダウンロードします（約30秒）。  
カレントディレクトリに `NotoSansCJKjp-Bold.otf` として保存されます。  
ダウンロード先は **GitHub 公式リポジトリ → jsDelivr CDN** の順にフォールバックするため、GitHub 障害時も高確率で取得できます。  
全 URL 失敗時はシステムフォントで代替します（日本語表示が乱れる場合があります）。

## Streamlit Community Cloud へのデプロイ

1. このフォルダを GitHub リポジトリとして push
2. [share.streamlit.io](https://share.streamlit.io) で `app.py` を指定してデプロイ
3. `requirements.txt` が自動で読み込まれます（API キー不要）
